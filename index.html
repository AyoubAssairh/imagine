Lightweight HTML5 Canvas Library

Add objects & images to canvas.

  • Drag & drop canvas elements including images.
  • Event handling for canvas elements.
  • Under 15KB minified, 4KB gzip.

How to build

You will need node installed as a development dependency. See node's site for help with that.

  1. npm install -g grunt-cli
  2. npm install -g bower
  3. npm install
  4. bower install
  5. Build - grunt (To view all grunt tasks, open Gruntfile.js)

Example Use

<!DOCTYPE html>
<html>
<head>
</head>
<body>
    <canvas id="canvas" width="300" height="300"></canvas>

    <script src="canvas.js"></script>
    <script>
        var canvas = new app.Canvas('canvas');
        var rect = new app.Rectangle();
        canvas.add(rect);
    </script>
</body>
</html>
